ความรู้เบื้องต้นเกี่ยวกับผังงาน

Flowchart (ผังงาน) คือ การใช้ภาพสัญลักษณ์เรขาคณิตแทนลำดับขั้นตอนการทำงาน ทบทวนสัญลักษณ์ก่อนเข้าเล่นเกม

เริ่มต้น / สิ้นสุด (Terminal)

ใช้จุดเริ่มต้นหรือจุดสิ้นสุดกระบวนการ

การปฏิบัติงาน (Process)

การประมวลผล คำนวณ หรือกำหนดค่า

รับ/แสดงข้อมูล (Input/Output)

นำเข้า/ส่งออกข้อมูลทั่วไปไม่ระบุอุปกรณ์

การตัดสินใจ (Decision)

ตรวจสอบเงื่อนไขเพื่อแยกทิศทางตามค่า จริง/เท็จ

แสดงผลจอภาพ (Display)

แสดงผลลัพธ์ออกทางหน้าจอคอมพิวเตอร์

รูปแบบผังงาน: แบบลำดับ (Sequence)

การทำงานเป็นเส้นตรงจากบนลงล่าง ไม่มีเงื่อนไขที่ซับซ้อน ตัวอย่างเช่น "กระบวนการชงกาแฟ"

เริ่มต้น (Start)
รับน้ำและต้มน้ำให้เดือด
ใส่กาแฟและคนให้เข้ากัน
สิ้นสุด (End)

รูปแบบผังงาน: แบบทางเลือก (Selection)

มีการแยกทิศทางการทำงานโดยอาศัยเงื่อนไขการตัดสินใจ ตัวอย่างเช่น "ตรวจสอบคะแนนสอบ"

Start
รับคะแนนสอบ (Score)
คะแนน >= 50?
ใช่ → แสดงผล "ผ่าน"
ไม่ใช่ → แสดงผล "ไม่ผ่าน"
End

รูปแบบผังงาน: แบบทำซ้ำ (Iteration)

วนทำงานในบล็อกเดิมซ้ำจนกว่าเงื่อนไขจะกลายเป็นเท็จ ตัวอย่างเช่น "การนับเลข 1 ถึง 5"

กำหนด i = 1
i <= 5?
ใช่
แสดงค่า i และเพิ่มค่า i++
(วนกลับขึ้นไปตรวจสอบเงื่อนไขใหม่)

การเขียนรหัสเทียม (Pseudo Code)

รหัสเทียมคือการจำลองขั้นตอนเขียนโค้ดคอมพิวเตอร์อย่างง่ายด้วยถ้อยคำภาษาอังกฤษหรือไทยที่เรียงลำดับอย่างเป็นตรรกะ

แบบลำดับ

BEGIN
  INPUT length
  area = length * length
  OUTPUT area
END

แบบทางเลือก

BEGIN
  IF score >= 50 THEN
    PRINT "Pass"
  ELSE
    PRINT "Fail"
  ENDIF
END
